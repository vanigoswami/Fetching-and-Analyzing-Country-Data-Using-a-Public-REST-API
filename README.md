
# Country Data Utility

A minimal Python script to fetch, process, and analyze live global data from the **REST Countries API**.

### **Features**
* **Fetch:** Requests live JSON data with error handling.
* **Process:** Parses names, population, region, and area into a **Pandas DataFrame**.
* **Analyze:** Identifies the most populated region and the largest country by area.

### **Requirements**
```bash
pip install requests pandas
```

### **Usage**
1. Open `main.ipynb`.
2. Run all cells to see the live summary.
